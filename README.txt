ANDREI TOMA - PREMIUM A4 CV

1. Upload index.html to your GitHub Pages repository.
2. Create an assets folder.
3. Add your photo as: assets/me.jpeg
4. Open the page in Chrome and click Print / Save PDF.
5. In print settings choose:
   - Destination: Save as PDF
   - Paper size: A4
   - Margins: None
   - Scale: 100% or Fit to page
   - Background graphics: ON

This version has the CSS inside index.html, so GitHub Pages cannot break the styling.
